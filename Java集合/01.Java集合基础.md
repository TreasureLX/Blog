### Java集合基础

1. Java集合框架总览

   集合框架是一个用户展现和操作集合的统一架构，其实现细节能够让集合独立地操作。

   基本组成

   * 集合接口 Collection、Map

   * 基础设施

   * 通用实现   ArrayList、LinkedList、Queue

   * 抽象实现  

     AbstractCollection、AbstractList、AbstractSet

   * 遗留实现 线程安全的集合Vector、Stack、Enumeration、BitSet
