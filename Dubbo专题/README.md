# DUBBO源码解析
dubbo在分布式系统中扮演着服务治理的角色，对各种服务进行管理，使得分布式系统能有条不紊的运行。虽然在工作中我运用了dubbo框架，但仅仅是在使用的层面，对其原理不是很了解。每次遇到到问题都不能快速的解决，于是决定阅读dubbo源码来了解其原理，学习dubbo框架的思想和设计，提高自己的眼界和水平。
